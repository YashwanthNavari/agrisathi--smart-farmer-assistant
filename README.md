## 📖 What is AgriSathi?

> **AgriSathi** *(meaning "Farmer's Friend" in Hindi)* is a **production-ready, AI-powered Progressive Web Application** built specifically for Indian farmers.

India has **140 million+ farming households**. Most face the same problems every season:
- 🦠 Crop diseases spread silently — detected too late
- 💸 Market prices are hidden behind layers of middlemen
- 📜 Government schemes exist but farmers can't navigate them
- 🌦️ Weather alerts don't reach the field in time

**AgriSathi solves all of this** — in one mobile-first app powered by **Google Gemini AI**, with a browser-based persistent state system that works even offline.

## ✨ Features — Full Breakdown

### 🩺 AI Crop Disease Detector
> Powered by **Google Gemini 1.5 Flash** multimodal vision

- 📸 Upload or take a photo of any affected crop part (leaf, stem, root)
- 🔬 AI performs tissue-level analysis and identifies the disease
- 💊 Returns a full diagnosis report with **organic AND chemical treatment plans**
- 🛒 Links directly to the Marketplace to buy treatment products
- 📤 Share or print the diagnosis report

---

### 🛒 Smart Marketplace
> Buy, sell, and rent — all in one place

- 🌱 Browse **Seeds, Fertilizers, Pesticides, Equipment**
- 🚜 Rent machinery like tractors and harvesters by the day
- 🔍 Search + Category filters (Supplies / Rental / Equipment)
- 📦 **Functional shopping cart** — add items, remove individually, clear all, subtotal calculation
- 💾 Cart is **persistent** — survives page refresh via localStorage
- 📈 **Live APMC Mandi rates** — real-time simulated price feed for Wheat, Soyabean, Cotton with trend indicators (Rising / Falling / Stable)
- 🏪 Sell directly to verified buyers — skip the middlemen

---

### 🤖 AI Expert Chat
> Conversational farming advisor powered by Gemini

- 💬 Ask questions in natural language — crop management, irrigation, pesticide dosage, soil health
- 🎙️ **Voice input support** — Sathi listens and transcribes
- 🤩 Animated "Sathi Listening..." wave visualizer during voice capture
- 📝 Responses rendered in beautiful **Markdown** — with headers, bullets, and tables
- ⏱️ Animated typing indicator (bouncing dots) while AI is processing
- 🕐 Message timestamps for every conversation
- 💾 Full conversation history maintained during session

---

### 🌱 Farm Management
> Precision lifecycle tracker for your crops

- 🗺️ Register and track multiple farm plots (persistent via localStorage)
- 📊 **Visual growth progress bars** per crop with percentage completion
- 🌡️ Real-time soil stats — Soil Health, Moisture %, Temperature, Protection status
- 🧪 **NutriGauge SVG circles** — animated circular gauges for Nitrogen (N), Phosphorus (P), Potassium (K) levels
- 📅 **Pro Action Log** — time-stamped farm activity timeline (done / pending / planned)
- 💰 **Estimated Yield Value** widget — net profit forecast with confidence score
- ➕ Register new plots via a modal form

---

### 📊 Pro Dashboard
> Command center at a glance

- 🌤️ **Weather Card** — current temperature, humidity %, wind speed in glassmorphism panel
- 📡 **SathiTicker** — scrolling live news/alert ticker (unseasonal rain, price spikes, crop alerts)
- 👁️ **Sathi Eye™ Field Monitor** — satellite AI field health viewer with field imagery, recalibrate action, and layer controls
- ⚡ **AI Intelligent Tasks** panel — quick-launch shortcuts to Disease Detector, Farm Plan, Market Insights
- 📈 Market pulse mini-widget — live Wheat (Sharbati) price from Nagpur APMC
- 🔗 One-tap Expert Chat shortcut card

---

### 🏛️ Government Schemes Browser
> Every scheme, simplified

- PM-Kisan, PMFBY, Kisan Credit Card, Soil Health Card, and more
- Filter by category — all Central and State agricultural schemes in one place
- Eligibility summary and apply links

---

### 🛒 Persistent Cart System
> Sessions remembered, nothing lost

- Cart is stored in **localStorage** — items survive page refresh
- Full cart drawer with slide-in animation
- Per-item remove button (hover to reveal)
- Subtotal auto-calculated for numeric-price items
- "Clear All" button in cart header
- Empty state with illustration

## ⚡ Getting Started Locally

### Prerequisites
- **Node.js 18+** — [nodejs.org](https://nodejs.org)
- **Google Gemini API Key** (free tier available) — [aistudio.google.com](https://aistudio.google.com)

### 1️⃣ Clone
```bash
git clone https://github.com/YashwanthNavari/agrisathi--smart-farmer-assistant.git
cd agrisathi--smart-farmer-assistant
```

### 2️⃣ Install dependencies
```bash
npm install
```

### 3️⃣ Configure environment
```bash
cp .env.example .env.local
```
Open `.env.local` and add your key:
```env
GEMINI_API_KEY=your_gemini_api_key_here
```

### 4️⃣ Start dev server
```bash
npm run dev
```
Open **http://localhost:3000/agrisathi--smart-farmer-assistant/** in your browser 🚀

### 5️⃣ Build for production
```bash
npm run build
```

## 🚀 Deploying to GitHub Pages

This project has **automated CI/CD** — just push to `main` and it deploys itself.

### First-time setup:
1. Go to **Repository → Settings → Pages**
2. Set source to **GitHub Actions**
3. Add your key: **Settings → Secrets and variables → Actions → New repository secret**
   - Name: `GEMINI_API_KEY`
   - Value: your Gemini API key
4. Push any change to `main` — the workflow triggers automatically ✅

### Workflow breakdown (`.github/workflows/deploy.yml`):
```
Push to main
    ↓
[Build Job — ~16s]
  ├── Checkout code
  ├── Setup Node 20
  ├── npm ci
  └── vite build → ./dist
    ↓
[Deploy Job — ~11s]
  └── actions/deploy-pages → GitHub Pages
    ↓
✅ Live at: https://yashwanthnavari.github.io/agrisathi--smart-farmer-assistant/
```
